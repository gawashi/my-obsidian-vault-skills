# スライド編集ガイド

このフォルダのスライドを**自分で**追加・修正・削除するための手引き。

## 0. まず起動と反映

- 起動：`present.cmd` をダブルクリック → Chrome が `http://localhost:PORT/index.html` を開く
- 編集の反映：`index.html` / `theme.css` を保存 → **Chrome を Ctrl+Shift+R（強制再読込）**。ビルド工程なし
- 発表者ビュー：Chrome で **S** キー → 別ウィンドウに「台本＋次スライド＋タイマー」（自分だけに見える）
- 全画面：**F** ／ 次へ：**→** ／ 戻る：**←** ／ 一覧：**Esc**

> ⚠️ 発表者ビュー（S）は `present.cmd`（ローカルサーバ）経由でないと動かない。`index.html` をダブルクリックの直開き（file://）では出ない。

## 1. 基本ルール：1スライド ＝ 1つの `<section>`

`index.html` の `<div class="slides"> … </div>` の中に `<section>…</section>` が並ぶ。**ファイルの上から順がスライド順**。

| やりたいこと | 操作 |
|---|---|
| 追加 | どれかの `<section>…</section>` を丸ごとコピペ → 置きたい位置に貼る → 中身を書換 |
| 修正 | その `<section>` の中の文字を直す |
| 削除 | `<section>` から `</section>` まで丸ごと消す |
| 並べ替え | `<section>` ブロックごと上下に移動 |
| 発表者ノート | `<section>` 内に `<aside class="notes">…</aside>`（聴衆には出ない） |
| 段階表示 | 要素に `class="fragment"` を付けると矢印キーで後から表示 |

## 2. コピペ用パーツ早見表

すべて `theme.css` のクラスを使うだけ。色・サイズは `theme.css` 側で一括管理。骨組みの `index.html` に各パーツの実例が入っているので、そこからコピーしてもよい。

### ふつうのスライド
```html
<section>
  <h2>タイトル</h2><hr class="rule">
  <p>本文。<b>強調</b> や <code>コード語</code> も使える。</p>
  <ul><li>箇条書き</li><li>箇条書き</li></ul>
  <aside class="notes">手元の台本。</aside>
</section>
```

### 2カラム ＋ 下のコールアウト
```html
<section>
  <h2>タイトル</h2><hr class="rule">
  <div class="cols">
    <div class="col"><h3>左の見出し</h3><p>…</p></div>
    <div class="col"><h3>右の見出し</h3><p>…</p></div>
  </div>
  <p class="callout">強調したい一文</p>            <!-- warn を足すと注意色（赤帯）: class="callout warn" -->
</section>
```

### リード文（大きめの一文）
```html
<p class="lead">大きく見せたい一文。</p>
```

### コードを見せる
```html
<pre class="code"><code>{
  "key": "value"
}</code></pre>
```
> `<code>` の中はそのまま表示される。`<`（小なり）を書きたいときだけ `&lt;` と書く。

### スクショ差し込み枠（当日キャプチャ用）
```html
<div class="placeholder">
  <div class="ph-title">🎬 スクショ枠：何の画面か</div>
  <div class="ph-spec">撮るべき内容・ファイルパスなど</div>
</div>
```
撮ったら枠ごと `<img src="assets/ファイル名.png" alt="説明">` に差し替える。

### ラベルバッジ＋見出し
```html
<div class="headrow"><span class="badge">ラベル</span><h2>見出し</h2></div>
```

### 表（比較）
```html
<table class="cmp">
  <tr><th>見出しA</th><th>見出しB</th></tr>
  <tr><td>…</td><td>…</td></tr>
  <tr class="root"><td>強調行</td><td>…</td></tr>   <!-- root で背景色付き -->
</table>
```

## 3. 守るべき注意点（これだけで崩れない）

- `<section>` と `</section>` は**必ずペア**で増減させる
- `<head>` の `<link>`、ページ末尾の `<script>`、`vendor/` フォルダ、`serve.mjs` は**触らない**（エンジン部分）
- 1枚に溢れたら**フォントを縮めず**、スライドを分割するか `fragment` で段階表示する

## 4. 文字サイズ・色を一括で変えたい

`theme.css` の上部を編集（`index.html` は触らない）：
- 本文サイズ：`.reveal p, .reveal li { font-size: 27px; }` の数値
- 見出し：`.reveal h2 { font-size: 46px; }` / タイトル `.reveal h1 { font-size: 76px; }`
- 色：`:root` の `--accent`（OMRON ブルー）`--head`（濃紺）`--bg`（背景）`--ink`（本文）`--warn`（注意の赤）など
> px はキャンバス 1920×1080 上の値。reveal が画面サイズに合わせて全体を等倍縮小する。

## 5. PDF に書き出したいとき（配布・保険）

Chrome で `http://localhost:PORT/index.html?print-pdf` を開く → 印刷 → 送信先「PDFに保存」→ 詳細設定で「背景のグラフィック」ON → 保存。

## 6. ファイルの役割

| ファイル | 役割 | 触る？ |
|---|---|---|
| `index.html` | 全スライドの内容 | ◎ ここを編集 |
| `theme.css` | 色・サイズ・レイアウト | ○ サイズ調整時 |
| `present.cmd` | 起動ランチャ | × |
| `serve.mjs` | ローカルサーバ | × |
| `vendor/` | reveal.js 本体 | × |
| `assets/` | スクショ置き場 | ○ 画像を入れる |
